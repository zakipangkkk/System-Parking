# 🚗 Sistem Parking Management

Sistem ini merupakan aplikasi manajemen parkir berbasis web dengan fitur autentikasi multi-role untuk mengatur akses pengguna berdasarkan peran (admin, petugas, dan owner).

## 🚀 Fitur Utama
- Login multi-role (Admin, Petugas, Owner)
- Manajemen data kendaraan
- Manajemen area parkir
- Transaksi masuk & keluar kendaraan
- Monitoring aktivitas parkir
